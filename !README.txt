This folder ("Trinity") contains everything for the triple course project for the second semester of School Year II.

Legend:
csweb - Web Design and Development
csspark - Database Systems
cspuff - Cloud Computing

The folder structure is as follows:

Folder "database" contains files for csspark, it has the SQL files required for the creation of the table as well as the files for querying the database and the tables contained therein. It will contain the required report for the csspark project and explanations and images used in the process of fulfilling project requirements.

Folder images contains images used in the website (csweb).

Folder pages has the .php files necessary to use the website (csweb)

Folder styles has the stylesheets for linking with the .php in the pages folder.
